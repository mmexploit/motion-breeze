# Agent Instructions

Follow the instructions in AGENTS.md when working with this repository.

## Agent skills

### Issue tracker

Issues and PRDs live as local markdown files in `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses five canonical roles: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo. Read `CONTEXT.md` at root and `docs/adr/` for domain language and architectural decisions. See `docs/agents/domain.md`.
